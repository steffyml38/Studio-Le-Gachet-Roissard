# Studio-Le-Gachet-Roissard
Studio cosy pour 2 à 4 personnes à Roissard, à 5 min à pied du Domaine de Messenas. Situé au calme, il offre un espace de vie avec canapé BZ, cuisine équipée, salle d’eau avec WC et chambre double en mezzanine. Idéal pour profiter d’un mariage ou d’un séjour nature, entre confort et tranquillité.
